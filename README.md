# BigQuery Data Ingestion Pipeline

## 📌 Project Overview
This project demonstrates an automated data pipeline to ingest local CSV datasets into **Google BigQuery**. It showcases core Data Engineering skills including Python scripting, Cloud Data Warehousing, and API authentication.

## 🛠️ Tech Stack
* **Language:** Python
* **Cloud Platform:** Google Cloud Platform (GCP)
* **Data Warehouse:** Google BigQuery
* **Libraries:** `google-cloud-bigquery`

## ⚙️ Setup & Installation
1. Clone the repository:
   `git clone https://github.com/longpvps37966/bigquery-pipeline.git`
2. Install the required library:
   `pip install google-cloud-bigquery`
3. Place your `credentials.json` and `global_health_data.csv` in the root folder.
4. Run the script:
   `python load_to_bigquery.py`
